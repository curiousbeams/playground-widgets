---
title: Observable Notebooks
---

Import [observable notebooks](https://observablehq.com/).

```md
:::{any:bundle} https://curiousbeams.github.io/esm-widgets/observable-notebook.js

{
  "notebook": "https://api.observablehq.com/@gvarnavi/ray-diagrams.js?v=4",
  "cells": [
    "viewof rayDiagram"
  ],
  "overrides": {
    "viz_width": "=width"
  }
}
:::
```
